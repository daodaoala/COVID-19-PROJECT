# COVID-PROGECT2

## 코로나 19 웹 사이트 
* 코로나 19는 수도권 거주자를 위하여 코로나 현황 및 다양한 서비스를 제공하는 반응형 웹 서비스
* 사용자가 원하는 자치구의 확진자 현황 및 선별 진료소, 선제검사 신청, 안전 수칙, 피해 지원 등을 확인할 수 있다.
* Node.js 서버를 연결하여 사이트 구축
